## SLIDE 6 — Course Curriculum

### What We Cover — Course by Course

#### Master Session 1 — How the Internet Works
- DNS, TCP/IP, HTTP, packets, status codes, URL anatomy
- How browsers render pages
- *Why:* You should understand the world your code lives in before writing a single line

#### Master Session 2 — Software Development Lifecycle (SDLC)
- Agile, Scrum, sprint planning, standups, retrospectives
- CI/CD, DevOps, team roles, burn-down charts
- *Why:* Professional engineers don't just write code — they follow a process

#### Course 1 — HTML *(Web Fundamentals)*
- Semantic structure, accessibility, ARIA, forms, validation
- *Outcome:* Write a valid, accessible HTML5 document from memory

#### Course 2 — CSS *(Web Fundamentals)*
- Selectors, cascade, box model, Flexbox, Grid, responsive design, animations, dark/light theming
- *Outcome:* Build any layout without a framework

#### Course 3 — JavaScript *(Web Fundamentals)*
- Variables, functions, arrays, DOM, events, async/await, Fetch API, localStorage, DevTools
- *Outcome:* Build a fully interactive app that fetches live data

#### Course 4 — Git & GitHub *(Web Fundamentals)*
- Commits, branching, pull requests, code review, CI pipelines
- *Runs alongside every other course*
- *Outcome:* Collaborate and version-control like a professional

#### Course 5 — React
- Components, hooks, routing, global state, TanStack Query, React Hook Form, Tailwind, shadcn/ui
- *Outcome:* Ship a production-grade single-page application

#### Course 6 — Frontend AI Development
- LLM fundamentals, streaming APIs (Claude & GPT-4o), image analysis, voice input, tool calling
- *Outcome:* Integrate real AI features into a React application — the differentiator that gets you hired

## SLIDE 7 — Mode of Learning

### How This Program Is Delivered

**This is not a YouTube playlist. This is a structured cohort.**

#### 1. Learning Management System (LMS)
- All lessons, notes, exercises, and resources are inside the LMS platform
- Track your progress lesson by lesson
- Submit exercises and get feedback
- Access your certificate of completion

#### 2. Ebook
- A complete program ebook covering all 8 courses
- Includes all code snippets, diagrams, tables, and exercises
- Yours to keep — read offline, print, annotate
- *"From Zero to AI-Powered Developer" — Samuel M. Ortil, 2026*

#### 3. GitHub Notes
- All course notes are version-controlled on GitHub
- You always have the latest version
- You can raise issues, suggest corrections, and fork the notes
- Learning Git happens here — by actually using it

#### 4. Learning Community (WhatsApp Group)
- This is where the cohort lives between sessions
- Ask questions, share progress, help each other
- Instructors are active here
- Accountability partners come from here

#### 5. Weekend Live Sessions
- Regular live classes over the weekend
- Real-time coding, Q&A, and problem-solving
- Recorded and available in the LMS for replay

#### 6. Workshops
- Hands-on project workshops — you build, we guide
- Work on real problems, get live feedback
- Simulates what it feels like to work on a team

#### 7. Master Sessions
- Delivered to the full cohort — everyone attends
- Deep dives: How the Internet Works + SDLC
- These are the sessions that separate people who *know code* from people who *think like engineers*
